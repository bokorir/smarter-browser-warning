Show warning popup for not supported browsers.

== Description ==

Many users are using outdated browsers. If you want to notify them and suggest up to date browsers this plugin is for you.

The plugin identifies the minimum browser version required for your site and pops up a warning message if the browser is outdated.

== Installation ==

1. Upload and extract the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
